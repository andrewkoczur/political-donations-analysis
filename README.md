# Political Donations Analysis
Analyzing small-dollar political donations using panel data and econometric methods

## Overview
This project analyzes small-dollar political donations using public FEC data. The goal is to understand how donation behavior varies across time, location, and economic conditions.

## Tools Used
- R
- tidyverse
- fixest

## Project Structure
- data/: raw and cleaned datasets
- scripts/: data cleaning and analysis code
- results/: tables and figures
- report/: final write-up

## Status
Project in progress.
